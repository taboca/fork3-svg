What is Fork3?

Fork3 is a Web-site creation tool that focus on graphics and style, and seems to be growing with some structural navigation framework to support certain *classes* of web sites. This project is under-construction and is also a research project.

Labs and Opportunities

Fork 3 has some different classes of visual behavior. You can use these classes in your markup and tie with classes of content, and in the future, classes of web services. This is a break-section element.

Featured item

This is a featured product or item you want to talk about. At this point the brek-section element is not dynamic. We may consider having some elements that associates JavaScript library elements to brings things on the FLY.

One of the cool things of Fork3 is that it wants to be friendly with search crawlers and yet to offer a rich, dynamic, and flexible Inner-Browsing framework where content from remote sources can be fetched and so on.
