This is Fork3

Welcome to Taboca Fork-3. The SVG-based, Open-source toolkit for graphics generation.

Fork 3 aims to support developers to generated graphics from SVG and XML-based data-sources. The toolkit wants to evolve to something powerful and flexible that allows end-users and developers to easily drop behavior to their graphic elemnts and create user interfaces with ease.

Hello Graphics!

Project Information

This page has some entry level ( and draft and early deployed ) documentation on the project. You can access the source-code at Fork3 Github project and please contribute to the project. If you want to compile the graphics you will need Apache Ant, Apache batik, the batik Rasterizer Ant task, and also a recent Java Development Kit.

Support the Project

You can support the project in many ways. Check in code, send public graphics tothe repository, create new SVG elements, support the HTML/CSS templates, and more. If you are not a developer you can also support the project using it on your Web site or project or also via donations:
